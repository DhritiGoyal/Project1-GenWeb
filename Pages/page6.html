my understanding and reflection

We followed the instructions and drew everyones patterns. We were the code. Creating these output. Interesting juxtaposition to us using machines to code for this entire semester and now becoming the code ourselves. Some turned out super interesting, but some were pretty confusing. I saw everyone just peeking over the person next to them to see how they are taking in the information. Vague language can be interpreted so differently by people-can create super interesting results.

How important is language will giving out instructions. How can vague language be used in favour of a project? Are there only certain scenarios, where vague language could work to create a system? For instance, an urban development system built on vague language could be a complete disaster.
